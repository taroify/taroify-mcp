## API

### Props

| 参数       | 说明                          | 类型               | 默认值     |
| ---------- | ----------------------------- | ------------------ | ---------- |
| type       | 类型，可选值为 `spinner`      | _string_           | `circular` |
| size       | 加载图标大小，默认单位为 `px` | _number \| string_ | `30px`     |
| direction | 排列方向，可选值为 `horizontal` | _string_ | `vertical` |
| children   | 加载文案    | _ReactNode_          | -    |
